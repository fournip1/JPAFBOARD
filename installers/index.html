For Debian linux distro

Download this file and then type in a terminal:
sudo dpkg -i jpafboard_2.5_amd64.deb

If you want to uninstall it, just run:
sudo apt remove jpafboard

For Windows 10 (64 bits)

Download this file, open it and follow the instructions.

For Mac OS

  1. Make sure that Java 17 is properly set on your computer by downloading and installing jdk 17.
  2. Download and double click on this jar file to run the application.

If it does not work, please check that you have downloaded the correct files for your operating system.
You may as well try the open with command and check that the jar file is correctly opened with Java 17.
